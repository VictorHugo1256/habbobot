# FenixBOT
> Discord with everything a Habbo Server needs.

## Installation

Windows:

```sh
Download NodeJs here: https://nodejs.org/en/download/
npm install discord.js --save
npm i isgd
npm i canvas
npm i mysql

node index.js
```
## Configuration

```sh
Edit config.js with your hotel link and name

```
